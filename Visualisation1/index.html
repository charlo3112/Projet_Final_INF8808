Comment lire: Chaque ligne est une chanson. Les couleurs indiquent le niveau de popularité.

Interaction: Cliquez dans un axe pour filtrer, puis déplacez les axes pour comparer des relations.

Objectif: Ce diagramme à coordonnées parallèles permet d'explorer les corrélations entre les attributs musicaux (danceability, energy, tempo, valence, liveness) et la popularité des chansons. Chaque ligne représente une chanson, colorée selon son niveau de popularité. L'objectif est d'identifier si certaines combinaisons de caractéristiques audio sont systématiquement associées aux chansons les plus populaires.
